# מסמך הדרכה למתנדבות — שפרה ופועה

ברוכות הבאות לאפליקציית שפרה ופועה! האפליקציה החדשה שלנו תעזור לכל מערך ההתנדבות לעבוד בצורה נוחה, מהירה ומסודרת יותר.

## 1. כניסה לאפליקציה (Login)
* גשו לכתובת האפליקציה (https://shifra-pua.vercel.app).
* הזינו את מספר הטלפון שלכן.
* קוד זיהוי (OTP) יישלח אליכן ב-SMS. הזינו אותו כדי להיכנס.

## 2. מזהה מתנדבת
לכל מתנדבת יש תפקיד שמשויך אליה מראש על ידי ההנהלה:
* **מבשלת (Cook)** - אחראית על הכנת הארוחות.
* **מחלקת (Driver)** - אחראית על איסוף והובלת הארוחות ליולדות.
* ניתן לקבל את שני התפקידים יחד.

## 3. מבשלות (Cooks)
כאשר את מחוברת כמבשלת, המסך הראשי יציג בפנייך את רשימת הארוחות הפתוחות:
1. **קבלת ארוחה**: לחצי על כפתור "אני אבשל!" כדי לשייך אלייך את הארוחה.
2. **פרטי יולדת**: תוכלי לראות אילו רגישויות או דרישות (לדוגמה, צמחוני, חריף) יש ליולדת.
3. **הארוחה מוכנה**: ברגע שהאוכל ארוז ומוכן לאיסוף, לחצי על "הארוחה מוכנה ✔️". פעולה זו תעביר את הארוחה למחלקות (Drivers).
4. **תפריט שבת**: בימי חמישי ושישי ישנו אזור מיוחד להזנת פריטי תפריט השבת שאתן מכינות. אל תשכחו למלא אותו!

## 4. מחלקות (Drivers)
כאשר את מחוברת כמחלקת, תראי רק ארוחות שכבר "מוכנות לאיסוף":
1. **לקיחת משלוח**: לחצי על "אני אקח את זה!" כדי לאשר שאת בדרך לאסוף את האוכל מהמבשלת.
2. **ניווט**: תוכלי להשתמש בכפתורי הניווט (Waze / Google Maps) שיופיעו כדי להגיע לכתובת היולדת.
3. **סיום מסירה**: לאחר שמסרת את האוכל ליולדת, לחצי על כפתור "הארוחה נמסרה 📦".

## 5. יולדות (Beneficiaries)
אם נרשמת כיולדת המקבלת ארוחות:
* תוכלי לראות את הארוחות שמתוכננות להגיע אלייך מראש.
* כאשר האוכל נמסר, תוכלי לאשר קבלה ("קיבלתי את האוכל").
* ניתן להשאיר הערות תודה ודירוג למתנדבות בסיום המסירה.

## 6. טיפים נוספים
* **סנכרון יומן**: לחצו על הקישור ה"הוסף ליומן" במסך שלכן (מבשלות ומחלקות) כדי להוריד את המשמרות שלכן ישירות ליומן Google/Apple שלכן!
* **התראות (Push)**: ניתן להפעיל קבלת הודעות והתראות מיידיות דרך "פרופיל האישי" שלכן באפליקציה (גם כשהטלפון סגור!).
* **מצב לילה**: האפליקציה תומכת במצב כהה או בהיר באופן אוטומטי בהתאם להגדרות המכשיר שלכן.
* **מסך הבית (PWA)**: ניתן להוסיף את האפליקציה ישירות למסך הבית של הטלפון כמו אפליקציה רגילה בלחיצה על "הוסף למסך הבית" דרך הדפדפן שלכן.
